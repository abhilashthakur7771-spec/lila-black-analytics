An interactive telemetry visualization tool built for Level Designers and Product Managers to identify map imbalances, analyze bot behavior, and optimize player retention.

## 🛠 Features
* **Global Death Heatmaps:** Aggregate death events across 100+ matches to find systemic map flaws.
* **Step-by-Step Playback:** Scrub through individual player journeys to see exactly where they engaged.
* **Automated Bot Detection:** Identify AI pathing vs. human movement via path entropy analysis.
* **Lobby Noise Warning:** Automatically identifies and flags pre-match lobby telemetry.
